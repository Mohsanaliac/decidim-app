# Decidim::Idealchat

This module will help me to add the decidim chatbot for the user support..

## Usage

Idealchat will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-idealchat"
```

And then execute:

```bash
bundle
```
